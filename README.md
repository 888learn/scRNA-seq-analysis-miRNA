# scRNA-seq-analysis-miRNA
# scRNA-seq analysis

## Overview
This repository contains code for single-cell RNA-seq analysis using Seurat.

## Requirements
- R (v4.3.3)
- Seurat (v4.4.0)
- ggplot2(3.5.2), dplyr(1.1.4), patchwork(1.3.0)，clustree(0.5.1)，cowplot(1.1.2)，qs2(0.1.5)，ggpubr(0.6.0)，harmony(1.2.3)，tidyverse,readr(2.1.5),tidyr(1.3.0),scDblFinder(1.16.0),presto(1.0.0),ggrepel(0.9.4),ggsignif(0.6.4),ggbeeswarm(0.7.2),data.table(1.14.10),stringr(1.5.1),

## Installation
Install required packages from CRAN/Bioconductor.

## Usage
Run scripts in the `code/` folder step by step.

## Demo
No separate demo dataset is provided. The full dataset (2.1GB) is available at Zenodo DOI: 10.5281/zenodo.19206091. See Instructions for Use for how to run the analysis.

## Reproducibility
All major figures in the manuscript can be reproduced using the provided scripts.
